The files in this directory are used to help run
the various utilities for working with the BioWarehouse.

Please read the documentation for these utilities 
at warehouse/doc/Utils.html.

Files
-----
build.xml - Warehouse master build file
warehouse-isntall.xml - Warehouse master install build file
developer.properties.template - properties for build files
runDeleteDataSet.sh - runs dataset deletion program
runSchemaDiffTool.sh - runs program to compare XML schemas and report differences
